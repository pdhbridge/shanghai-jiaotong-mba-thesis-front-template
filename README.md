# 上海交通大学MBA论文样式导入工具

本项目提供VBA代码，用于自动导入上海交通大学MBA学位论文所需的样式和编号格式。

## 功能特点

- ✅ 自动创建论文所需的所有样式（标题、正文、图表、公式等）
- ✅ 多级编号系统：
  - 章标题：第一章、第二章（中文数字）
  - 节标题：1.1、1.2（阿拉伯数字）
  - 小节标题：1.1.1、1.1.2（阿拉伯数字）
  - 图/表编号：图1、表1（全文连续）
  - 公式编号：(1-1)、(2-1)（按章重置）
  - 分项序号：（1）、（2）（中文括号）
  - 小项序号：①、②、③（圈号）
- ✅ 正确的大纲级别配置
- ✅ 符合上海交通大学MBA论文格式要求

## 使用方法

### 方法一：直接导入VBA代码

1. 打开Microsoft Word 2021
2. 按 `Alt + F11` 打开VBA编辑器
3. 右键点击项目名称 -> 导入文件
4. 选择 `上海交大MBA论文样式导入.bas` 文件
5. 返回Word，按 `Alt + F8` 运行宏
6. 选择 `导入论文样式` 并执行

### 方法二：使用加载项（推荐）

1. 打开Word，点击「开发」选项卡
2. 点击「加载项」-> 「Word加载项」
3. 点击「添加」按钮
4. 选择本项目中的 `.bas` 文件
5. 样式将自动加载
